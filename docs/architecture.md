# Architecture

```
                         ┌─────────────┐
                         │   Client    │
                         └──────┬──────┘
                                │
                         ┌──────▼──────┐
                         │   Gateway   │ :8000
                         └──────┬──────┘
            ┌───────────────────┼───────────────────┐
            │                   │                   │
     ┌──────▼──────┐     ┌──────▼──────┐     ┌──────▼──────┐
     │    Auth     │     │   Product   │     │    Order    │
     │   Service   │     │   Service   │     │   Service   │
     │    :8001    │     │    :8002    │     │    :8003    │
     └──────┬──────┘     └──────┬──────┘     └──────┬──────┘
            │                   │                   │
            └───────────────────┼───────────────────┘
                                │
                    ┌───────────┴───────────┐
                    │                       │
             ┌──────▼──────┐         ┌──────▼──────┐
             │ PostgreSQL  │         │    Redis    │
             └─────────────┘         └─────────────┘
```

## Services

| Service | Port | Description |
|---------|------|-------------|
| Gateway | 8000 | API Gateway, JWT 검증, 라우팅 |
| Auth | 8001 | 회원가입, 로그인, 토큰 관리 |
| Product | 8002 | 상품, 재고, 핫딜 관리 |
| Order | 8003 | 주문 생성, 조회, 취소 |
